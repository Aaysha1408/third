# Restaurant Website

## Project Overview
This project is a Restaurant Website built using HTML and CSS that showcases a restaurant’s brand, menu, and services.  
The website focuses on clean UI design, structured layout, and responsiveness, making it easy for users to explore menu items and restaurant details.

## Features
- Attractive landing section with restaurant branding
- Menu section displaying food categories and items
- About section describing the restaurant
- Contact section with basic details
- Responsive layout for different screen sizes
- Clean and well-organized code structure

## Technologies Used
- HTML5– for semantic structure
- CSS3 – for styling, layout, and responsiveness
